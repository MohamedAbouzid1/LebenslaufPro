# LebenslaufPro

A professional German CV (Lebenslauf) builder with AI-powered evaluation, built for the German job market.

![LebenslaufPro Preview](./public/preview.png)

## Features

- 🎯 **Single Optimized Template** - ATS-friendly German CV format
- 📝 **Live WYSIWYG Editor** - Real-time preview as you type
- 🤖 **AI Evaluation** - Get instant feedback on your CV quality
- 💾 **Draft Management** - Save, version, and manage multiple CVs
- 📊 **ATS Score** - Ensure your CV passes applicant tracking systems
- 🌍 **Bilingual Support** - German UI with Arabic tooltips
- 📱 **Responsive Design** - Works on all devices
- 🔒 **Privacy-First** - GDPR compliant, no sensitive data by default

## Tech Stack

- **Frontend**: Next.js 14 (App Router), TypeScript, Tailwind CSS, shadcn/ui
- **Backend**: Next.js API Routes, Prisma ORM, PostgreSQL
- **Auth**: NextAuth.js with email/password and Google OAuth
- **PDF Export**: React PDF Renderer
- **State Management**: Zustand
- **Testing**: Vitest + React Testing Library

## Getting Started

### Prerequisites

- Node.js 18+
- PostgreSQL database
- Google OAuth credentials (optional)

### Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/lebenslauf-pro.git
cd lebenslauf-pro
```
